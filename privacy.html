Privacy Policy

Last updated: 2026-07-20

This application ("the App") is a personal automation tool operated by a single developer to generate and publish video content to the developer's own TikTok account via the TikTok Content Posting API.

Data Collected

The App stores a TikTok OAuth access token and refresh token, obtained via TikTok's official login flow, in order to publish videos on behalf of the developer's own TikTok account. No data about other TikTok users is collected, stored, or processed.

Data Use

Stored tokens are used only to authenticate API requests to TikTok on behalf of the developer's own account. Video files are generated locally and uploaded directly to TikTok; they are not shared with any third party.

Data Storage

All tokens and generated media are stored locally on the developer's own machine and are not transmitted to any third-party service.

Data Sharing

No data is sold or shared with third parties.
